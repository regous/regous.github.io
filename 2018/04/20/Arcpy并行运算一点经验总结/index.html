Arcpy并行运算一点经验总结

这段时间帮一个同事做一个分幅裁切的东西。

实现调用的是ArcGIS的Split和Append\Copy\FeatureClassToFeatureClass GP工具。分完一层再分一层，然后再处理分完成的数据。其实这种情况用多线程完成可以提高处理的速度。于是就想看看能不能用多线程来处理完成这个操作。

Arcpy不支持多线程

测试代码如下：

import arcpy
import threading

def List(wkp):
    arcpy.env.workspace = wkp
    dss = arcpy.ListDatasets()
    if dss is None:
        arcpy.AddError('ListThread arcpy.ListDatasets return None')
        return
    else:
        arcpy.AddMessage('ListThread arcpy.ListDatasets return {} dataset'.format(len(dss)))

    dss.insert(0, None)
    fcs = []
    for ds in dss:
        temp_fcs = arcpy.ListFeatureClasses(feature_dataset=ds)
        fcs.extend(temp_fcs)

    arcpy.AddMessage('ListThread arcpy.ListFeatureClasses return {} FeatureClass'.format(len(fcs)))

class ListThread(threading.Thread):
    def __init__(self, wkp):
        threading.Thread.__init__(self)
        self.workspace = wkp

    def run(self):
        List(self.workspace)


if __name__ == '__main__':
    wkp_0 = arcpy.GetParameterAsText(0)
    wkp_1 = arcpy.GetParameterAsText(1)
    if not wkp_0:
        wkp_0 = r'E:\GamingSpace\FunAddin\Demo\Points.gdb'
    List(wkp_0)
    t = ListThread(wkp_0)
    t.start()
    t.join()

输出结果如下：

arcpy.ListDatasets return 0 dataset
arcpy.ListFeatureClasses return 1 FeatureClass

Process finished with exit code -1073741819 (0xC0000005)

第一个执行List函数时是正常输出的，但是当第二次执行时，进程崩溃了，0xC0000005代码应该是访问错误的地址。

修改代码，变更List的执行顺序

结论

Arcpy不支持多线程，可能是因为arcpy.env的限制。

多进程

测试代码：

import arcpy
import multiprocessing


def List(wkp):
    arcpy.env.workspace = wkp
    dss = arcpy.ListDatasets()
    if dss is None:
        arcpy.AddError('arcpy.ListDatasets return None')
        return (None, None)
    else:
        arcpy.AddMessage('arcpy.ListDatasets return {} dataset'.format(len(dss)))

    dss.insert(0, None)
    fcs = []
    for ds in dss:
        temp_fcs = arcpy.ListFeatureClasses(feature_dataset=ds)
        fcs.extend(temp_fcs)

    arcpy.AddMessage('arcpy.ListFeatureClasses return {} FeatureClass'.format(len(fcs)))
    return (len(dss), len(fcs))


if __name__ == '__main__':
    wkp_0 = arcpy.GetParameterAsText(0)
    wkp_1 = arcpy.GetParameterAsText(1)
    if not wkp_0:
        wkp_0 = r'E:\GamingSpace\FunAddin\Demo\Points.gdb'
    if not wkp_1:
        wkp_1 = r'E:\GamingSpace\FunAddin\Demo\Polygons.gdb'

    pool = multiprocessing.Pool()
    jobs = []
    jobs.append(pool.apply_async(List, (wkp_0,)))
    jobs.append(pool.apply_async(List, (wkp_1,)))
    print(jobs[0].get())
    print(jobs[1].get())

输出结果如下：

arcpy.ListDatasets return 0 dataset
arcpy.ListFeatureClasses return 1 FeatureClass
(1, 1)
arcpy.ListDatasets return 0 dataset
arcpy.ListFeatureClasses return 1 FeatureClass
(1, 1)

脚本不能在进程中运行

Run Python script in process 选项不能勾上，否则就会启动另外两个Arcmap进程

Always run in foreground 勾不勾都行，不影响。
