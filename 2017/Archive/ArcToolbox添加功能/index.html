ArcToolbox添加功能

ArcToolbox添加的功能可以分两类：Script、Tool。

Script：用Arcpy写，调用的时候其实就是调用Python的脚本。

Tool：用AO写成，需要注册。

Arcpy

先从简单的开始

一个加法的例子

先新建一个Toolbox

添加一个Script

勾选 Store relative path names (instead of absolute path) ，让Toolbox以相对路径的方式存储Python脚本的位置。

设置脚本的参数，传入参数较为简单，注意返回值，在Tool里是作为传出参数，所以Parameter Properties里的Direction值为Output，而传入参数为Input，这是比较容易被忽略的地方，如果没有设置传出参数，那么脚本的调用是不会传出结果的。

从Geoprocessing-Results可以查看执行结果

一个获取要素类空间坐标系的例子

# -*- coding:utf-8 -*-

import arcpy
import os
import sys

def get_spatial_reference():
    dataset = arcpy.GetParameter(0)
    arcpy.AddMessage(dataset)
    spatial_ref = arcpy.Describe(dataset).spatialReference
    print(spatial_ref)
    arcpy.SetParameter(1,spatial_ref)


if __name__=='__main__':
    get_spatial_reference()

在加法的例子中，使用 arcpy.SetParameter(3, res) 设置传出参数，注意这个语句的索引是 3 ，而在空间参考的例子中索引是 1。两个例子分别使用了 arcpy.GetParameterAsText 和 arcpy.GetParameter 来获取脚本的传入参数。在 ArcCatalog 界面上调用GP工具时，所有的参数都以字符串的方法展示给调用者。有些参数无法从字符串转化为对象（比如Spatial Reference），就要用 arcpy.GetParameter 来获取相关的参数。

再添加一个工具类，实现 IGPFunctionFactory 接口

[
    Guid("182AB4E8-B408-466F-B5F9-9B9B58BDC782"),
    ComVisible(true)
]
public class ThePacificOcean:IGPFunctionFactory
{
    // Register the Function Factory with the ESRI Geoprocessor Function Factory Component Category.
    #region "Component Category Registration"
    [ComRegisterFunction()]
    static void Reg(string regKey)
    {
        GPFunctionFactories.Register(regKey);
    }

    [ComUnregisterFunction()]
    static void Unreg(string regKey)
    {
        GPFunctionFactories.Unregister(regKey);
    }
    #endregion
    #region IGPFunctionFactory Members
    public string Alias
    {
        get { return "太平洋"; }
    }

    public UID CLSID
    {
        get
        {
            UID id = new UIDClass();
            id.Value = this.GetType().GUID.ToString("B");
            return id;
        }
    }

    public IGPFunction GetFunction(string Name)
    {
        switch (Name)
        {
            case ("PlayFunction"):
                IGPFunction gpFunction = new PlayFunction();
                return gpFunction;
        }

        return null;
    }

    public IEnumGPEnvironment GetFunctionEnvironments()
    {
        return null;
    }

    public IGPName GetFunctionName(string Name)
    {
        IGPName gpName = new GPFunctionNameClass();

        switch (Name)
        {
            case ("PlayFunction"):
                return (IGPName)CreateGPFunctionNames();
        }
        return null;
    }

    public IEnumGPName GetFunctionNames()
    {
        IArray nameArray = new EnumGPNameClass();
        nameArray.Add(CreateGPFunctionNames());
        return (IEnumGPName)nameArray;
    }

    public string Name
    {
        get { return "The Pacific Ocean"; }
    }
    #endregion

    #region 创建Function Name
    public IGPFunctionName CreateGPFunctionNames()
    {
        PlayFunction aPlayFunction = new PlayFunction();
        IGPFunctionName functionName = new GPFunctionNameClass();
        IGPName name = functionName as IGPName;

        name = (IGPName)functionName;
        name.Category = "Happy Fun";
        name.Description = "乘风破浪";
        name.DisplayName = aPlayFunction.DisplayName;
        name.Name = aPlayFunction.Name;
        name.Factory = this;
        return functionName;
    }
    #endregion
}
